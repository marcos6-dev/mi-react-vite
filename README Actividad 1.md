
# READMI SEMANA 1 

- NODE

Es un entorno de ejecución de código abierto para JavaScript, diseñado para ejecutarse del lado del servidor y no en el navegador. Se basa en el motor V8 de Google Chrome, permitiendo desarrollar aplicaciones web rápidas, escalables y en tiempo real gracias a su arquitectura asíncrona y basada en eventos.

- ¿Qué es npm?

es el gestor de paquetes predeterminado para Node.js, considerado el registro de software más grande del mundo para JavaScript.Permite a los desarrolladores instalar, compartir y gestionar bibliotecas (dependencias) reutilizables, automatizando el desarrollo y evitando escribir código desde cero. Se instala automáticamente con Node.js.

- ¿Cómo se ejecuta el proyecto?

Para ejecutar el proyecto, primero debes abrir una terminal dentro de la carpeta del proyecto y asegurarte de haber instalado las dependencias con npm install (solo la primera vez). Después, inicia el servidor de desarrollo con npm run dev, lo que levantará la aplicación en una dirección local, normalmente http://localhost:5173/. Al abrir esa URL en el navegador podrás ver el proyecto funcionando y cualquier cambio que guardes en el código se actualizará automáticamente.

- ¿Qué es un componente?

Un componente en React es una pieza independiente y reutilizable de la interfaz, construida como una función que devuelve JSX (una mezcla de JavaScript y HTML). Los componentes permiten dividir la aplicación en partes más pequeñas, organizadas y fáciles de mantener. En este proyecto se creó un componente llamado Saludo, que simplemente devuelve un texto y se integra dentro del componente principal mediante la etiqueta <Saludo />.
